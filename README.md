# Discrete_Math_Computer_Project

Комп'ютерний проєкт з курсу "Дискретна математика" (UCU). Репозиторій містить повну імплементацію та порівняльний аналіз алгоритмів пошуку найкоротшого шляху в розріджених графах.

## Про проєкт

У цьому проєкті реалізовано та досліджено ефективність двох алгоритмів для задачі Single-Source Shortest Path (SSSP):
1.  **Класичний алгоритм Дейкстри** (власна реалізація на основі бінарної купи).
2.  **Новітній алгоритм (Duan et al., 2025)**, описаний у статті *"Breaking the Sorting Barrier for Directed Single-Source Shortest Paths"*.

Основна увага приділена порівнянню швидкості роботи цих алгоритмів на розріджених графах (sparse graphs) різного розміру та щільності.

## Реалізований функціонал

* **Алгоритм Дейкстри:** Оптимізована версія з використанням `heapq` та матриць суміжності `NumPy`.
* **Алгоритм Duan et al. (2025):** Імплементація експериментального алгоритму для орієнтованих графів.
    Обмеження?
* **Генератор графів:** Скрипт для створення тестових даних з гарантованими шляхами та контрольованою щільністю (density).
* **Бенчмаркінг:** Автоматизоване тестування швидкості виконання обох алгоритмів.
* **Візуалізація:** Побудова графіків залежності часу виконання від кількості вершин ($V$) та щільності графа.
* ?**Валідація:** Перевірка коректності результатів через еталонну реалізацію в бібліотеці `NetworkX`.

## Технології

* **Python 3.10+**
* **NumPy** — для високоефективних матричних обчислень.
* **NetworkX** — використовується виключно для генерації графів та валідації результатів (вбудовані алгоритми пошуку не використовуються в основній логіці).
* **Matplotlib** — для візуалізації результатів експериментів.

## Структура репозиторію

* `algorithms/`
    * `dijkstra.py` — реалізація алгоритму Дейкстри.
    * `duan_algo.py` — реалізація алгоритму Duan et al.
* `data_generator.py` — генерація .csv файлів з розрідженими графами.
* `benchmark.py` — скрипт для запуску тестів порівняння продуктивності.
* `main.py` — точка входу для запуску пошуку на конкретному файлі.
* `results/` — збережені графіки та логи експериментів.
